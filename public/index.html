Paired Up

A pair-programming app that helps you connect and code with awesome tech mentors through Video Chat, a Live Code Editor, and Direct Messaging

Project Summary
Learning how to code? Need a Partner? PairedUp is a way to connect with awesome TechMentors. If your looking to learn a particular programming language then PairedUp will match you with fellow coders who can pair program with you.


Tech Stack
AngularJS, Node/Express, Socket.io, MongoDB, Passport.js, OAuth2

Contributions
I owned the private messaging feature by building a RESTful API layer with Node/Express and Websockets. Writing the frontend code in AngularJs allowed for a modular codebase, enabling new features to be added quickly. My greatest achievement with the application was creating a frontend architecture that increased the application’s performance, communicated well with the backend, and enhanced user experience.

AfricaExpress

An open-source marketplace for independent merchants

Project Summary
Africa express is a digital marketplace for artist to sell handcrafted goods and for shoppers to discover one-of-a-kind pieces. The application includes a simple backend for sellers to upload product images and custom buy links (such as Paypal). Products are posted to the community marketplace for easy purchasing.


Tech Stack
AngularJS, Node/Express, MongoDB

Contributions
I created the original application in Ruby on Rails; however I later refactored the codebase to work with javascript MEAN stack technologies to improve performance by 30%. I created a responsive frontend with Angular Material design to improve user experience and implemented a RESTful API layer using Node and Express for client-side data retrieval.

TweetWatch

Monitor your favorite Twitter profiles in real-time

Project Summary
TweetWatch is a simple dashboard that displays the most recent tweets from a given profile. Users have the ability to reorganize profile sections with the applications simple drag and drop feature.


Tech Stack
ReactJS, BackboneJS, Node/Express, Javascript

Contributions
I leveraged Twitter's API to collect the latest tweets from each profile. I used BackboneJS to organize to manage incoming tweet collections and ReactJS to create view components. I had the most fun implementing the the applications drag and drop feature.

QuickWork

Project Summary
QuickWork makes it easy for teachers to create new assignments and monitor their students progress. Students have the ability to submit multiple assignments as they get feedback on their work.


Tech Stack
AngularJS, MaterializeCSS, Node/Express, MongoDB, Passport.js

Contributions
One key feature that I implemented was user authorization using Passport.js local strategy and OAuth2. User sessions were managed using JSON web tokens. I also introduced privacy settings that are only available to online user.
